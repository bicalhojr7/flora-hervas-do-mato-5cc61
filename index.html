Dúvidas Frequentes

Vocês trabalham com reservas?

Sim, operamos exclusivamente sob reserva para garantir a qualidade dos ingredientes frescos colhidos no dia. O contato é feito via WhatsApp.

O menu possui opções veganas?

Sim, nossa filosofia "Ervas do Mato" prioriza elementos vegetais. Temos um menu degustação 100% plant-based disponível sob consulta.

Qual o horário de atendimento?

Atendemos para jantar de Quinta a Sábado, das 19h às 23h. Domingos para almoço das 12h às 16h.
